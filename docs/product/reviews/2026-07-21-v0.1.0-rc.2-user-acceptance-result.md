## Observed acceptance defects

The RC.2 root-layout correction is accepted as an engineering improvement, but the exact RC.2
product is not accepted because independent desktop use found four product defects:

1. choosing run purpose **Assessment** prevented an otherwise technically ready engineering run;
2. the eVTOL application icon appeared inside the window but was missing from the Windows
   taskbar in the released layout;
3. Model Studio exposed task deactivation but no clear global **Delete node** action;
4. press-and-hold followed by pointer movement did not visibly move a node.

## Required correction

The replacement candidate must keep `formal_run_authorized=false` as scientific provenance while
allowing a technically ready `purpose=assessment` run with a visible warning. It must resolve the
published icon from the executable directory, expose staged global node deletion with explicit
confirmation and task-scheme cascade semantics, and make press-and-hold drag work with actual
WinUI pointer routing.

RC.2 remains immutable. Its tag, ZIP, checksum, manifest and verification report continue to
describe the exact package the user tested; no later fix may be attributed to RC.2.
